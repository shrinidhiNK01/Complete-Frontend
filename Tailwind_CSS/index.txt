Tailwindcss:- Framework to style the webpage by adding classes to the HTML elements faster
* We can use tailwindcss only in html file

google:- tailwind css:- docs:-play CDN:- copy the script line <script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script> and paste it in html head section

For Intellisense:-
=======================
1)Install Node
2)npm install -D tailwindcss postcss autoprefixer vite
3)npx tailwindcss init -p
4)install tailwind css intellisense extension
5)add * in content:[*] in tailwind.config.js
6)add scripts in package.json:- {"start":"vite","build":"vite build"} inside "scripts" section 
7)create main.css and insert and link
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
8)open terminal and run 
    npm run start
9)create index.html and link the main.css
